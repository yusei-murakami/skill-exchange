# 個人のスキルを交換し困りごとを解決したり、新しい学びを得る。


## 主な機能
### 1. スキル登録
- タイトル  
- 説明  
- 経験年数  
- レベル設定

### 2. 検索
- キーワード検索

### 3. スキル診断
- 自分のスキルの強みと弱みを可視化  
- レーダーチャートの分析表示

### 4. コメント
- 個々のスキルページにコメントが可能  
- 質問や相談のやりとりができる

### 5. ログイン/ログアウト
- ユーザーネームととパスワード認証    

### 6. ダッシュボード
- 登録したスキルの一覧  
- コメント履歴
